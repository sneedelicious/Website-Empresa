
HF Reis - Soluções em Organização de Documentos 📂
Este projeto é um website institucional desenvolvido para a disciplina de Aplicações Web do curso de Ciência da Computação na Universidade Veiga de Almeida (UVA). O objetivo é apresentar os serviços de encadernação, digitalização e indexação de documentos para cartórios e empresas.

🚀 Funcionalidades
Interface Responsiva: Adaptável para dispositivos móveis e desktops via Media Queries.

Menu Hamburguer: Navegação otimizada para dispositivos menores.

Formulário de Contato.

Design Profissional: Layout focado em usabilidade e identidade visual corporativa.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização avançada, uso de Flexbox para layout e animações de hover.

JavaScript: Lógica para funcionamento do menu mobile e alertas do formulário.

🎨 A interface utiliza uma paleta de cores baseada em tons de azul (darkblue e blue), transmitindo confiança e segurança, elementos essenciais para o ramo de organização de documentos e cartórios.
